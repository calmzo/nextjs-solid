## 介绍

本文档的这一部分向您展示如何使用 Tailwind CSS 模板和 Bootstrap 模板配置联系表单。

## 使用 PHP 联系表单：

对于 Bootstrap 模板，我们过去提供一个只能用于 PHP 服务器的 PHP 脚本，

如果 `phpmailer` 功能正常运行。

它的工作原理是：只需从以下位置**将电子邮件地址替换为您自己的**

```bash
assets->php>contact.php
```

如果服务器端一切配置正确，此过程应该可以正常工作。

## 使用 FormBold：通用表单 API（推荐）：

[FormBold](https://formbold.com/) 是一个强大的表单 API 和无服务器表单后端，适用于所有托管、SSG 和框架。

我们强烈建议使用无服务器端点（[FormBold](https://formbold.com/)）来使联系表单动态化。因为现在大多数用户不使用 PHP 服务器。

以下是如何将 [FormBold](https://formbold.com/) API 与您的联系表单连接：

**步骤#1**：在 [FormBold](https://formbold.com/) 上创建账户并创建表单。

**步骤#2**：使用此代码将 API 集成到您的联系表单中。确保您的表单方法设置为 **POST**

```html
<form action="https://formbold.com/s/unique_form_id" method="POST">
  <input type="email" placeholder="email" name="email" />
  <input type="text" placeholder="subject" name="subject" />
  <textarea name="message" placeholder="message"></textarea>

  <button type="submit">Send Message</button>
</form>
```

您可以在 [FormBold](https://formbold.com/) 仪表板的 **设置>集成** 下找到 `unique_form_id`。

将其复制并粘贴到 **action** 属性中。

这只是基本设置，FormBold 有很多很棒的功能，比如用于垃圾邮件保护的 re-captcha，以及与其他应用程序（如 Webhook、Slack、Notion 和 GoogleSheet）集成的能力。
在[这里](https://formbold.com/)了解更多关于 [FormBold](https://formbold.com/) 的信息。
