## **介绍**

本文档涵盖了如何安装和使用 Tailwind CSS 模板的说明，并回答您在使用模板时可能遇到的任何问题和问题。

如果您有任何超出本文档范围的问题，请随时[提交支持工单](https://uideck.com/support/)或在[Discord](https://pimjo.com/discord)上联系我们。

我们会尽快回复您。

Tailwind 模板使用纯 HTML 和 Tailwind CSS 以及一些 Vanilla JS 脚本构建。我们使用 Alpine JS 来实现滑块、视频和图片弹出等交互功能。

## **下载模板：**

您可以通过访问您的账户下载大包中所有可用的模板。

只要您在我们这里有一个账户，您就可以随时下载它。

## **目录结构：**

当您下载模板并解压时，您会得到一个如下所示的文件夹：

```json
|-- 模板文件夹
    |-- src
    |    |-- css
    |       |-- 所有 CSS 文件
    |    |-- js
    |       |-- 所有 JS 文件
    |    |-- images
    |       |-- 所有图片
    |    |-- partials
    |       |-- HTML 公共部分，如页眉、页脚等
    |    |-- index.html 和所有 HTML 页面文件
    |
    |
    |-- 所有配置文件
```

如果您想自定义模板，只需转到 `src` 文件夹，您会找到需要自定义的所有文件。

## **安装模板：**

要使用 Tailwind 模板，首先我们必须安装模板中包含的依赖项和插件。

**注意**：在安装插件之前，请确保您的机器上安装了 Node.js。否则，命令将不起作用。

以下是您安装模板需要采取的步骤：

**步骤#1：** 从 GrayGrids 下载模板。当您下载模板时，您会得到一个 zip 文件。

**步骤#2**：解压/提取 zip 文件。提取后，您会找到所需的所有文件和资源。

**步骤#3**：现在在项目文件夹中打开终端/命令行。并运行此命令来安装插件和依赖项：

```jsx
npm install
```

安装需要几秒钟。安装完成后，运行此命令来启动项目：

```jsx
npm run start
```

当它启动时，它会自动在 `[localhost:3000](http://localhost:3000)` 上打开项目。

现在您可以更改默认内容并添加您自己的内容，并立即在浏览器中看到更改。

## **部署到服务器**

要部署模板，您必须生成 `build` 文件夹。您可以通过运行此命令来做到这一点：

```jsx
npm run build
```

它会为您提供一个 `build` 文件夹，现在您只需将其上传到您的服务器，您的网站就会上线。
