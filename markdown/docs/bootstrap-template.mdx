## 下载模板：

您可以通过访问您的账户下载大包中所有可用的模板。

只要您在我们这里有一个账户，您就可以随时下载它。

## **安装模板**：

为了能够使用模板，首先，您必须安装它。以下是您需要采取的安装步骤：

**步骤#1：** 从 GrayGrids 下载模板。当您下载模板时，您会得到一个 zip 文件。

**步骤#2：** 解压/提取 zip 文件。提取后，您会找到所需的所有文件和资源。

**步骤#3：** 现在在您的代码编辑器中打开文件夹，并根据您的需要更新内容。

**步骤#4：** 更新内容后，现在转到您的 cPanel 或打开您的 FTP 客户端（如 Filezilla）并将模板上传到您的服务器。确保将其上传到根目录。

安装完成，现在转到 **yourdomainname.com/index.html**。您应该看到网站正在运行。

## **目录结构**

当您提取模板文件夹时，您会在文件夹内看到类似这样的内容：

```
|-- 模板文件夹
    |-- assets
    |    |-- css
    |       |-- 所有 CSS 文件
    |    |-- scss
    |       |-- 所有 SCSS 文件
    |    |-- js
    |       |-- 所有 JS 文件
    |    |-- fonts
    |       |-- 所有字体文件
    |    |-- images
    |       |-- 所有图片
    |-- index.html 和所有 HTML 文件
```

## **自定义**

### **更改默认主题颜色**

如果您想更改默认主题颜色，请打开 `assets->css>main.css` 文件。找到默认颜色并用您喜欢的颜色替换它。

**注意：** 如果您熟悉 Scss，可以通过更改 Scss 变量来更改颜色。

### **更改默认字体**

就像颜色一样，如果您想更改默认字体，请转到 `assets->css>main.css`

找到所有 font-family 然后用您自己的 font-family 替换其属性。

确保在您的 HTML 或 CSS 文件中包含字体的 URL。否则，它将不起作用。

**注意：** 如果您熟悉 Scss，可以通过更改 Scss 变量来更改 font-family。
