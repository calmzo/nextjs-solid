## 基本使用和下载

GrayGrids 通过全访问计划提供必要的 Tailwind UI 组件。您可以通过将它们复制粘贴到您现有的 Tailwind 项目中来使用它们，或者您可以用它们构建全新的项目。

您也可以从您账户的下载部分下载离线文件。

## 配置：

我们可以在现有的 Tailwind 项目中使用 GrayGrids，也可以从头开始。我们将引导您完成这两个过程。

### 1. 从头开始

我们将向您展示如何从头开始使用 GrayGrids 项目。我们首先将设置一个 Tailwind CSS 项目，然后我们将在其中包含 GrayGrids。

以下是您从头开始 GrayGrids 项目可以采取的步骤：

**步骤#1**：安装 Tailwind 并生成配置文件。

```sh
npm install -D tailwindcss
npx tailwindcss init
```

**步骤#2**：安装 TailGrids

```sh
npm i tailgrids
```

**步骤#3**：使用 GrayGrids 配置更新配置文件。

```js
module.exports = {
  content: ["./*.html", "./ecommerce/*.html", "./assets/**/*.js"],
  theme: {},
  variants: {
    extend: {},
  },
  plugins: [require("tailgrids/plugin")],
};
```

**步骤#4**：将 Tailwind CSS 指令添加到您的 CSS 中。在项目的根目录或 **src** 目录中创建一个名为 **input.css** 的 CSS 文件。然后在文件顶部包含此代码。

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

**步骤#5**：使用构建命令生成 CSS 文件。

要做到这一点，首先我们必须将脚本添加到 **package.json** 文件中。确保您提供了 **input.css** 文件的正确路径。如果它在根目录中，请保持脚本不变，如果它在 **src** 下，请在斜杠前包含它。

```json
"scripts": {
    "build": "npx tailwindcss -i ./input.css -o ./dist/output.css --watch"
  },
```

然后我们必须运行构建命令：

```sh
npm run build
```

**步骤#6**：在 HTML 文件中包含编译的 CSS 文件。我们已在 **dist** 文件夹中编译了文件。

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <link href="/dist/output.css" rel="stylesheet" />
  </head>
  <body>
    <h1 class="text-3xl font-bold underline">Hello world!</h1>
  </body>
</html>
```

现在您只需从 GrayGrids 复制粘贴组件并构建网站。
